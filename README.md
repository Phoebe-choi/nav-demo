## 导航网站

这是一个漂亮的导页面
#### 主要功能
1. 键盘按键 + 鼠标点击 跳转相应网站功能。
2. 点击修改按钮 
3. 点击没有定义网站的按钮，会弹出设置界面。

#### 实现效果
1.键盘按键
- 如果有网址，跳转
- 如果没有网址，弹出对话框
- 点击页面上的按钮

2.如果有网址，跳转
- 如果没有网址，弹出对话框
- 点击页面上按钮的 edit button，弹出对话框，进行修改网址

3.弹框确认时，可以点击确认，也可以点击 Enter 按键

- 输入网址正确，储存到本地（localStorage）并跳转
- 输入网址错误，输入框清空 + 提示
- 弹框取消，重置弹框的信息 + 弹框隐藏
